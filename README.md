# bankist-app

Bankist application is fictional minimalistic banking application created from the Javascript course "The Complete JavaScript Course 2022: From Zero to Expert!" by Jonas Schmedtmann
#
In Bankist application user can:

 * Log in with a username and password
 * View and sort their monetary movements
 * Transfer money to another account
 * Request a loan from the bank
 * Delete their account

Registration form and procces is implemented so there is no need to use only hardcoded accounts
